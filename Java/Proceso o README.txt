1. Abren el CMD

2. Se mueven hasta donde esté la carpeta de la 
matriz.java y matriz.in, ponen en el cmd: javac Matriz_2x2.java

3. Con esto les generará un archivo .class

4. Luego en el cmd ponen: java Matriz_2x2
java Matriz_2x2 es el código pero sin extención

5. Para cambiar de matriz, cambian en el archivo = new File ("Matriz_2x2.txt");, y ponen el txt de la matriz

EXTRA:
Pueden poner tanto el .java como el txt en un proyecto que también les servirá
