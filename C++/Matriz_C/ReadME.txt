- Tienen que dejar la primera línea de cada .txt en blanco para que lea la matriz completa
(ver los txt para que entiendan)

- Para compilar archivos c++ (.cpp) tienen que descargar una herramienta, les dejo el link del 
video para que sigan los pasos para compilar correctamente archivos c++ (.cpp)

-------> https://www.youtube.com/watch?v=-e8Rgc9mBaY <-------

- Para compilar en cmd se dirigen hasta la ruta donde esta la carpeta del .exe, y solo escriben
el nombre del .exe (Ejemplo: en mi caso utilizo el comando 
"cd C:\Users\edand\OneDrive\Escritorio\Edwin\Matriz_C"
y estando en el directorio solo escribo "MatrizC" sin escribir el ".exe" y se ejecuta)

- Deben cambiar manualmente desde un editor de texto/código el nombre de los archivos donde
están las matrices .txt en la línea 14
